What is SwitchSetup?

This software is designed to configure network devices using Ansible, assisting configurations of many equipment.

Feel free to contribute to the project.

Advantage

The main advantage of this Software, if you are on a network with access to the equipment

you will be able to configure it through a cell phone for example.

How Work

To establish a connection with the equipment, you will need the SSH credentials, including the username and password.

Once connected, you can access the configuration interface where you

provide the relevant user and password information,

specify the equipment (Host) you want to access, and define the commands to be executed (Playbook).

How we build this

The ApiWeb was developed using the Django Framework and ansible libraries,

leveraging the power of Python and software engineering principles.
